# Examples

A small, self-contained worked example vault (one client hub + entity + an Office mirror + a repo
mirror) is on the roadmap.

For now:
- `docs/quickstart.md` walks through creating and populating a vault.
- `docs/methodology.md` shows the intended shape (MOC hubs, entity pages, mirrors).
- The pattern was extracted from a real small-business vault, which serves as the dogfooding
  reference.
