# Fieldkit Control

Synthetic Python package used by Vaultwright to exercise repository mirroring without relying on a
live external GitHub repository.

## Purpose

Fieldkit Control simulates a small automation-cell controller library. It is intentionally tiny, but
it includes enough README and documentation structure to make the generated repo mirror readable.

## Modules

- `fieldkit.control` - validates simple inspection readings and returns a routing decision.
- `docs/control-loop.md` - explains the fictional control-loop assumptions.
